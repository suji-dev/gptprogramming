C++ 학습 페이지

C++ 소개

C++은 다목적 프로그래밍 언어로, 시스템 프로그래밍, 게임 개발, 모바일 앱 등 다양한 분야에서 사용됩니다.

C++이란?

C++은 C 언어를 기반으로 한 객체지향 프로그래밍 언어입니다. 높은 성능과 풍부한 기능을 제공합니다.

왜 C++을 선택해야 할까요?

C++은 하드웨어 제어, 성능 최적화, 대규모 시스템 개발 등에 적합한 강력한 언어입니다.

C++의 난이도

C++은 다소 복잡한 언어이며, 입문자에게는 도전적일 수 있습니다. 그러나 그만큼 강력한 기능을 제공합니다.

필요한 프로그램

C++을 학습하려면 C++ 컴파일러와 통합 개발 환경(IDE)를 설치해야 합니다.

20가지 C++ 정보

  • C++은 1979년 Bjarne Stroustrup에 의해 개발되었습니다.
  • C++은 C 언어를 기반으로 하며, 객체지향 프로그래밍 개념을 도입했습니다.

C++ 예제 (10개)

3. 정수 연산

#include <iostream>
using namespace std;
int main() {
int a = 5;
int b = 3;
int result = a + b;
cout << "덧셈 결과: " << result << endl;
return 0;
}

4. 조건문 활용

#include <iostream>
using namespace std;
int main() {
int age = 20;
if (age >= 18) {
cout << "성인입니다." << endl;
} else {
cout << "미성년자입니다." << endl;
}
return 0;
}

5. 반복문 활용

#include <iostream>
using namespace std;
int main() {
for (int i = 0; i < 5; ++i) {
cout << "반복 횟수: " << i << endl;
}
return 0;
}

6. 배열 사용

#include <iostream>
using namespace std;
int main() {
int numbers[] = {1, 2, 3, 4, 5};
cout << "배열 요소: " << numbers[2] << endl;
return 0;
}

7. 함수 정의

#include <iostream>
using namespace std;
void greet(string name) {
cout << "안녕하세요, " << name << "님!" << endl;
}
int main() {
greet("홍길동");
return 0;
}

8. 클래스 정의

#include <iostream>
using namespace std;
class Circle {
public:
double radius;
double getArea() {
return 3.14 * radius * radius;
}
};
int main() {
Circle myCircle;
myCircle.radius = 5.0;
cout << "원의 넓이: " << myCircle.getArea() << endl;
return 0;
}

9. 파일 입출력

#include <iostream>
#include <fstream>
using namespace std;
int main() {
ofstream outputFile("example.txt");
outputFile << "파일에 쓰는 예제입니다." << endl;
outputFile.close();
return 0;
}
